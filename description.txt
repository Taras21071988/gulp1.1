// Описание отдельных команд для запуска 
    1 - gulp html --- запуск сборки файла index.html

    2 - gulp sass --- компиляция и сборка стилей

    3 - gulp images --- копирования изображений в папку сборки готового проекта

    4 - gulp server --- запуск сервера 

    5 - gulp clean --- удаление папки dist

    6 - gulp watch --- слежение за изменениями в файлах

    7 - gulp js --- работа с файлами js(Для корректной работы с js файлами, при подключение новых файлов нужно сделать описание этого файла в webpack.config.js)
